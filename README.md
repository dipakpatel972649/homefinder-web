# 🏠 JustHome – Real Estate Website UI

A modern and responsive **real estate website frontend** built using **HTML, CSS, and Bootstrap**.  
This project showcases property listings, search functionality, and a clean user interface design.

## 📌 Features

- 🔍 Property search bar (location / name input)
- 🏡 Featured property listings
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- 🧭 Navigation bar with dropdown menus
- 🏷️ Property tags (For Sale, Featured)
- 📦 Reusable card components
